## 📖 درباره پروژه

**TAAKAA-XI** یک پنل مدیریت کانفیگ مدرن، قدرتمند و کاملاً بهینه‌شده است که با هدف ساده‌سازی فرآیند تولید، مدیریت و توزیع کانفیگ‌های VLESS، Trojan و VMess طراحی شده است.

این پنل با استفاده از **Cloudflare Workers**، **KV** و **D1** ساخته شده و امکاناتی مانند محدودیت حجم و زمان، احراز هویت ادمین، مشاوره آفلاین بر اساس اپراتور و پرچم‌های منطقه‌ای را ارائه می‌دهد.

## ✨ ویژگی‌های برجسته

### 🎯 قابلیت‌های هسته
| ویژگی | توضیح |
|-------|-------|
| 🎯 **مشاوره آفلاین** | پیشنهاد بهترین تنظیمات بر اساس اپراتور کاربر بدون نیاز به اینترنت |
| 🌍 **پرچم منطقه‌ای** | نمایش کشور و پرچم روی هر کانفیگ به صورت خودکار |
| 🔐 **امنیت کامل** | ورود با رمز عبور و احراز هویت ادمین با قابلیت تغییر |
| 📊 **مدیریت کاربران** | محدودیت حجم و زمان برای هر کاربر به صورت مجزا |
| ⚡ **پروتکل‌های پیشرفته** | پشتیبانی کامل از VLESS / Trojan / VMess |
| 🛡️ **تکنیک‌های عبور** | Fragment / ECH / WARP / WARP Pro |
| 💾 **ذخیره‌سازی دوگانه** | KV + D1 برای امنیت و پایداری بیشتر |
| 🎨 **رابط کاربری مدرن** | طراحی شیشه‌ای، ریسپانسیو و کاملاً فارسی |

### 🔧 قابلیت‌های مدیریتی
- **مدیریت چندکاربره**: ایجاد، ویرایش، حذف و غیرفعال‌سازی کاربران
- **محدودیت ترافیک**: تنظیم حجم مصرفی روزانه و کلی
- **انقضای خودکار**: تاریخ انقضا برای هر کاربر به صورت مجزا
- **گزارش‌گیری**: مشاهده آمار مصرف و فعالیت کاربران
- **پشتیبان‌گیری**: خروجی و ورودی فایل تنظیمات به صورت JSON
- **همگام‌سازی**: قابلیت همگام‌سازی با نودهای دیگر

### 🚀 قابلیت‌های فنی
- **خروجی چندفرمته**: پشتیبانی از Clash, Sing-box, Base64
- **اسکنر آی‌پی**: پیدا کردن بهترین آی‌پی‌های تمیز
- **بهینه‌سازی خودکار**: کد مبهم‌سازی شده برای امنیت بیشتر
- **پشتیبانی از WebSocket**: اتصال پایدار و سریع
- **فال‌اوور هوشمند**: تغییر خودکار به آی‌پی جایگزین در صورت قطعی

## 🛡️ تکنیک‌های عبور از فیلترینگ

| تکنیک | کاربرد | وضعیت |
|-------|--------|--------|
| **Fragment** | تکه‌تکه کردن ترافیک برای دور زدن DPI | ✅ فعال |
| **ECH** | مخفی‌سازی کامل SNI و هدر درخواست | ✅ فعال |
| **WARP** | مسیریابی ترافیک UDP و بهبود سرعت | ✅ فعال |
| **WARP Pro** | نسخه‌ی حرفه‌ای و پایدارتر با سرعت بالاتر | ✅ فعال |
| **TLS Fingerprint** | شبیه‌سازی امضای مرورگرهای واقعی | ✅ فعال |
| **WebSocket Path Obfuscation** | مخفی‌سازی مسیر اتصال | ✅ فعال |

## 🚀 نحوه راه‌اندازی

### روش اول: Deploy to Cloudflare (ساده)
> 🔜 به زودی اضافه می‌شود

### روش دوم: نصب دستی

1. **کلون کردن مخزن**
```bash
git clone https://github.com/tentayzm/TAAKAA-XI.git
cd TAAKAA-XI
```

2. **نصب وابستگی‌ها**
```bash
npm install -g wrangler
```

3. **تنظیم D1 Database**
```bash
wrangler d1 create taakaa-db
```

4. **تنظیم متغیرهای محیطی**
```bash
wrangler secret put TAAKAA_DB
```

5. **دیپلوی روی Cloudflare**
```bash
wrangler deploy
```
