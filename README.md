## 💎 معرفی پروژه

**TAAKAA-XI** یک پنل مدیریت کانفیگ مدرن و قدرتمند است که با هدف ساده‌سازی فرآیند تولید و مدیریت کانفیگ‌های VLESS، Trojan و VMess طراحی شده است.

این پنل با استفاده از **Cloudflare Workers**، **KV** و **D1** ساخته شده و امکاناتی مانند محدودیت حجم و زمان، احراز هویت ادمین، مشاوره آفلاین بر اساس اپراتور و پرچم‌های منطقه‌ای را ارائه می‌دهد.

---

## ✨ ویژگی‌های برجسته

| ویژگی | توضیح |
|-------|-------|
| 🎯 **مشاوره آفلاین** | پیشنهاد بهترین تنظیمات بر اساس اپراتور کاربر |
| 🌍 **پرچم منطقه‌ای** | نمایش کشور و پرچم روی هر کانفیگ |
| 🔐 **امنیت کامل** | ورود با رمز عبور و احراز هویت ادمین |
| 📊 **مدیریت کاربران** | محدودیت حجم و زمان برای هر کاربر |
| ⚡ **پروتکل‌های پیشرفته** | VLESS / Trojan / VMess |
| 🛡️ **تکنیک‌های عبور** | Fragment / ECH / WARP / WARP Pro |
| 💾 **ذخیره‌سازی دوگانه** | KV + D1 برای امنیت بیشتر |
| 🎨 **رابط کاربری مدرن** | طراحی شیشه‌ای، ریسپانسیو و فارسی |

---

## 🛡️ تکنیک‌های عبور از فیلترینگ

| تکنیک | کاربرد |
|-------|--------|
| **Fragment** | تکه‌تکه کردن ترافیک برای دور زدن DPI |
| **ECH** | مخفی‌سازی کامل SNI |
| **WARP** | مسیریابی ترافیک UDP |
| **WARP Pro** | نسخه‌ی حرفه‌ای و پایدارتر |
